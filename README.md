# 📊 Employee Attrition Dashboard Analysis

## 📌 Project Overview
Employee attrition is one of the biggest challenges faced by organizations.  
High attrition leads to increased hiring costs, reduced productivity, and loss of experienced employees.

This project analyzes employee attrition patterns using **Excel Dashboard & Data Analysis techniques** to uncover insights about:

- Department attrition
- Gender distribution
- Employee trends
- Workforce structure

---

## 🎯 Objective

The objective of this project is to:

✔ Analyze employee attrition patterns  
✔ Identify departments with high attrition  
✔ Study workforce distribution by gender  
✔ Build an interactive dashboard for decision making  

---

## 🛠 Tools Used

- Excel
- Data Cleaning
- Pivot Tables
- Dashboard Design
- Data Visualization
